# Creating a Bug

Bug Entry Steps:

1. Close the ticket used to track your triage/investigation work
2. Create a DATAPLAT bug
3. Using the ticket template provide the necessary information. @Ticket Template
4. Assign the ticket to yourself
5. Assign the reporter to Garrett
6. Clone the ticket to the appropriate fwTeam
7. Assign the ticket to the appropriate fwTeam dev manager and leave a comment asking for the ticket to be assigned to a developer on the team.
