# Looker Tool

What is it?

This is a tool that lets us visualize data within bronze++ using a small set of filters we can modify. We can download and share this data with others in the form of excel docs.

This is used to collect and showcase data in a graph-like depiction. It can help you see weaknesses in deploys and even identify bugs in specific builds and environments. It is good for high leveled information and visualization

Who owns it?

This tool was originally created by Ty Henningsen, but I believe the maintenance of the tool has shifted. I now believe the maintenance is under Ray Hechavarria's purview, specifically being handled by Yatharth Malhortra.
